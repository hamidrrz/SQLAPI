Getting Started

1. Configure Database Connection

Navigate to the Settings page to configure your MySQL database connection.

2. Set Up Query Slots

Define your SQL queries in the 10 available query slots.

3. Execute Queries via API

Use the REST API to execute your queries with parameter support:

curl -X POST "https://your-domain.com/api/query/0" \
  -H "Authorization: Basic base64(username:password)" \
  -H "Content-Type: application/json" \
  -d '{"parameters":{"param1":"value1"}}'
